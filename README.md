# Background Changer

A simple React app that changes the full screen background color when you click a color button.

## Features

- Click a color button to change the background
- Smooth color transition effect
- Buttons fixed at the bottom of the screen

## Tech Stack

- React
- Vite
- Tailwind CSS

## Getting Started

Clone the repository and run:

```bash
npm install
npm run dev
```

Then open your browser at `http://localhost:5173`
